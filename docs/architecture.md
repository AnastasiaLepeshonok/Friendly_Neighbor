# Архитектура системы

## Общая структура проекта

Система представляет собой чат-бота для социальной сети ВКонтакте, предназначенного для координации локальной взаимопомощи между пользователями.

Проект состоит из следующих компонентов:

* модуль взаимодействия с VK API;
* модуль обработки пользовательских команд;
* модуль работы с базой данных;
* модуль обработки геолокации;
* система хранения заявок.

## Принцип работы системы

Пользователь взаимодействует с ботом через сообщения ВКонтакте. После получения команды бот обрабатывает запрос, обращается к базе данных и возвращает пользователю необходимую информацию.

При создании заявки данные пользователя сохраняются в базе данных. Заявки могут содержать текстовое описание, категорию помощи и географические координаты.

## Используемые технологии

* Python
* VK API
* SQLite

## Архитектурная схема

Система использует клиент-серверный подход:

Пользователь → VK API → Чат-бот → База данных

## Особенности архитектуры

* модульная структура;
* возможность расширения функционала;
* независимость компонентов;
* централизованная обработка запросов.
